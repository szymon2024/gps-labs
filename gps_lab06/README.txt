EN:
=====================================================================
          MATCHING EPHEMERIDES TO GPS SATELLITE OBSERVATIONS
=====================================================================

The program matches GPS ephemerides from a RINEX 3.04 NAV file to
satellite observations from a RINEX 3.04 OBS file. It process only GPS
satellite data.

Input
-----
  - RINEX 3.04 navigation file name
  - RINEX 3.04 observation file name

Output
------
  - GPS observations with attached navigation
    records containing ephemerides
    
Print of run
------------
Total observations: 7161
Without matched ephemeris: 0


PL:
=====================================================================
         DOPASOWANIE EFEMERYD DO OBSERWACJI SATELITARNYCH GPS
=====================================================================

Program znajduje efemerydy GPS z pliku RINEX 3.04 NAV dla obserwacji
satelitarnych z pliku RINEX 3.04 OBS. Program przetwarza tylko dane
satelitów GPS.

Wejście
-------
  - nazwa pliku nawigacyjnego RINEX 3.04
  - nazwa pliku obserwacyjnego RINEX 3.04

Wyście
------
  - obserwacje GPS z dołączonymi rekordami
    nawigacyjnymi zawierającymi efemerydy
    
Wydruk uruchomienia
-------------------
Total observations: 7161
Without matched ephemeris: 0
