EN:
=====================================================================
          MATCHING GPS EPHEMERIDES TO SATELLITE OBSERVATIONS
=====================================================================

The program finds GPS ephemeris from a RINEX NAV file for satellite
observations from a RINEX OBS file. The program only processes GPS
satellite data.

Input
-----
  - RINEX 3.04 navigation file name
  - RINEX 3.04 observation file name

Output
------
  - list of observations with matched
    navigation recrods (ephemerides)
    
Print of run
------------
Total observations: 7161
Without matched ephemeris: 0


PL:
=====================================================================
         DOPASOWANIE EFEMERYD GPS DO OBSERWACJI SATELITARNYCH
=====================================================================

Program znajduje efemerydy GPS z pliku RINEX NAV dla obserwacji
satelitarnych z pliku RINEX OBS. Program przetwarza tylko dane
satelitów GPS.

Wejście
-------
  - nazwa pliku nawigacyjnego RINEX 3.04
  - nazwa pliku obserwacyjnego RINEX 3.04

Wyście
------
  - lista obserwacji z dopasowanymi
    rekordami nawigacyjnymi (efemerydami)
    
Wydruk uruchomienia
-------------------
Łącznie obserwacji: 7161
Bez dopasowanej efemerydy: 0
